# v0.1.0 — 첫 공개 출시

## 새 기능

- **GitHub PR URL 입력만으로 한국어 PR 리뷰** — Claude AI가 자동 분석
- **두 가지 인증 방식**:
  - Claude Code Max 구독 (추가 비용 0)
  - Anthropic API 키 (종량제, PR당 ₩200~500)
- **사용자 PC에서 모든 처리** — 사용자 코드는 외부 서버로 절대 나가지 않음
- **OS keychain 안전 저장** — Windows Credential Manager / macOS Keychain
- **GitHub Personal Access Token** — 비공개 PR 지원 (선택)
- **다크모드** + Pretendard 한국어 폰트

## 다운로드

- Windows: `*.msi` — EV 코드 서명 (JCG Inc.), SmartScreen 즉시 OK
- macOS: `*.dmg` — 현재 unsigned (첫 실행 시 "어쨌든 열기" 클릭)
- Linux: `*.AppImage`

## 시스템 요구사항

- Windows 10 22H2+ / 11
- macOS 12+
- Ubuntu 22.04+ / Debian 12+
